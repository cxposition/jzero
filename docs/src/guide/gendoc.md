icon: ph:markdown-logo-bold

---

:::tip 目前仅支持 proto 可描述文件
:::

## markdown

```shell
jzero gen doc
```

默认输出路径 `desc/docs/markdown/index.md`

## json

```shell
jzero gen doc --format json
```

默认输出路径 `desc/docs/json/index.json`

## html

```shell
jzero gen doc --format html
```

默认输出路径 `desc/docs/html/index.html`
